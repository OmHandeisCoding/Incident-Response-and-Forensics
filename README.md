# Incident Response & Digital Forensics Portfolio

> A practical, auditor-accurate portfolio demonstrating end-to-end Incident Response (IR) and Digital Forensics capabilities — structured around **NIST SP 800-61r2** and real-world SOC/DFIR workflows.

---

## 🎯 What This Repository Demonstrates

- **Incident Response Lifecycle** — Detection through post-incident review
- **Threat-Specific Playbooks** — Phishing, Ransomware, Insider Threat
- **Digital Forensics Methodology** — Windows, Linux, Memory, and Disk forensics
- **Evidence Handling** — Chain of custody, integrity verification, documentation
